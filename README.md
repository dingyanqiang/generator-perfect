## 安装

```bash
npm install -g yo
npm install -g generator-perfect
```

创建Project:
```bash
yo perfect
```
创建Page:
```bash
yo perfect:page
```
创建Component:
```bash
yo perfect:component
```
创建Util:
```bash
yo perfect:util
```
